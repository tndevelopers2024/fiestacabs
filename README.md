# Fiesta Cabs — Corporate & Premium Mobility Solutions

> Designed and developed by Mohan.

Fiesta Cabs (Fiesta Smart Mobility) is India's premier corporate mobility and car rental service provider, delivering safe, punctual, eco-friendly, and seamless transportation solutions for enterprises, employee commuting, shuttle services, outstation rides, airport transfers, and luxury wedding rentals across major Indian cities (Chennai, Bangalore, Delhi, Gurgaon, Hyderabad, Mumbai, Pune, Kolkata, etc.).

This project is a modern, high-performance web application built using Next.js 16 (App Router), React 19, Tailwind CSS 4, Swiper, React Icons, and integrated server actions for email and lead management.

## Tech Stack Comparison

**Original Site Stack (Legacy):**
- **CMS:** WordPress
- **Page Builder:** Elementor / Legacy PHP
- **Database:** MySQL
- **Fonts:** Custom Fonts
- **Analytics:** Google Tag Manager

**New Redesign Stack (Modern):**
- **Framework:** Next.js 16 (App Router)
- **UI Library:** React 19
- **Styling:** Tailwind CSS 4 + Custom Responsive Design System
- **Language:** TypeScript
- **State & Context:** React Context API (`ModalContext`)
- **Sliders & Carousels:** Swiper.js
- **Icons:** React Icons
- **Email & Security:** Nodemailer & Google reCAPTCHA
- **Integrations:** Zoho SalesIQ Live Chat & Google Analytics
- **Document Processing:** pdf2json & pdf-parse

## Getting Started

First, install the dependencies:

```bash
npm i
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
